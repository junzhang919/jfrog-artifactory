# RubyGems Repositories
 - https://www.jfrog.com/confluence/display/RTF/RubyGems+Repositories
 - [bug] https://www.jfrog.com/jira/browse/RTFACT-13635  (fixed at 6.11.0)

## push gem package by grab
```
grab rt config gems
grab rt u gems demo-0.0.1.gem
```

## install
```
grab rt config gems-download
gem install demo
```
